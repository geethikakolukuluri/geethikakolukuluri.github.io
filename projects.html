Selected Engineering Projects

Cloud-Based IoT Healthcare Monitoring Simulation

Mar 2025 – Dec 2025

I designed and implemented a Java-based simulation framework to model real-time SPO2 patient monitoring as a latency-sensitive healthcare application using CloudSim Plus and EdgeCloudSim. By converting continuous time-series healthcare datasets into fixed time-window simulation tasks (cloudlets), I modeled realistic backend workloads to study client-server and edge-cloud interaction patterns. I implemented task execution and scheduling logic based on specific VM resources, including MIPS, CPU cores, memory, and bandwidth, to evaluate and analyze application-level latency, throughput, and concurrency behavior.

Java, CloudSim Plus, EdgeCloudSim, Distributed Systems Simulation, Client-Server Architecture, Performance Analysis.

SimpleFirebaseApp – iOS Cloud-Backed Application

Nov 2025

I developed a native iOS application using Swift and Xcode that demonstrates cloud-backed data storage and retrieval through integration with Firebase Cloud Firestore, focusing on application-to-backend integration. I designed a multi-screen UI using Storyboards, featuring form-based data input and a table-based display screen for viewing stored records with real-time synchronization via Firestore snapshot listeners. This project provided hands-on experience in integrating mobile clients with centralized cloud services, reflecting the interaction patterns found in enterprise-level platforms.

Swift, Xcode, Firebase, Cloud Firestore, iOS Storyboards, UITableView, Backend Integration.
